🚀 AOSP HAL Demo: Educational Android System Programming

Educational demonstration of Android system programming concepts through JNI/HAL integration. Features two complementary learning projects: FlashTune (pure JNI/HAL) and PulsePlay (hybrid approach) for understanding Android internals.
  • FlashTune: Pure JNI/HAL LED control demonstration without Android APIs - direct sysfs interface interaction for educational purposes
  • PulsePlay: Hybrid haptics implementation showing both educational JNI/HAL path and functional Android API usage
  • Educational Architecture: MainActivity → LedManager → JNI Bridge → HAL → sysfs interface
  • JNI Integration: Native method declarations and library loading with proper C++ bridging
  • HAL Concepts: Hardware abstraction principles with CMake build system integration
  • Learning Focus: Understanding Android system programming trade-offs and architectural patterns
  • Build System: Android NDK with CMake for native library compilation and JNI integration
